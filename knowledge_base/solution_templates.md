# Solution Templates & Problem-Solving Strategies

## General Problem-Solving Framework
1. **READ** carefully - identify what is given and what is asked
2. **CLASSIFY** - identify the topic and sub-topic
3. **PLAN** - choose the appropriate method/formula
4. **EXECUTE** - apply the method step by step
5. **VERIFY** - check if answer is reasonable and satisfies constraints
6. **COMMUNICATE** - state the answer clearly with units

## Template: Quadratic Equation Problem
Given: ax² + bx + c = 0
1. Compute discriminant D = b² - 4ac
2. If D > 0: two real roots x = (-b ± √D) / 2a
3. If D = 0: one root x = -b/2a
4. If D < 0: complex roots
5. Verify by substituting back
6. Check if problem asks for sum/product of roots

## Template: Probability Problem (Urn/Cards/Dice)
1. Identify sample space and total outcomes
2. Identify favorable outcomes
3. Apply P = favorable/total (for equally likely outcomes)
4. Check if events are dependent/independent
5. Apply appropriate formula (addition, multiplication, conditional)
6. Verify: answer must be between 0 and 1

## Template: Optimization Problem (Calculus)
1. Define variables and write objective function f(x)
2. Find domain restrictions
3. Compute f'(x) and set equal to 0
4. Solve for critical points
5. Use f''(x) to classify: f'' < 0 → max, f'' > 0 → min
6. Evaluate at critical points and endpoints
7. State the maximum/minimum value

## Template: Limits Problem
1. Try direct substitution first
2. If 0/0 form: factor and cancel OR L'Hôpital
3. If ∞/∞ form: divide by highest power OR L'Hôpital
4. If trig: use standard limits (sin x/x → 1)
5. If exponential: use (1+1/n)^n → e

## Template: Matrix Equation AX = B
1. Compute det(A)
2. If det(A) ≠ 0: X = A⁻¹B (unique solution)
3. Compute A⁻¹ using adjugate: A⁻¹ = adj(A)/det(A)
4. Multiply A⁻¹ × B
5. Verify by computing AX and checking equals B

## Template: Bayes' Theorem Problem
1. List all hypotheses H₁, H₂, ..., Hₙ (mutually exclusive, exhaustive)
2. State prior probabilities P(Hᵢ)
3. State likelihoods P(E|Hᵢ)
4. Compute P(E) = Σ P(E|Hᵢ)·P(Hᵢ)
5. Apply Bayes': P(Hᵢ|E) = P(E|Hᵢ)·P(Hᵢ) / P(E)

## Template: Arithmetic/Geometric Series
1. Identify first term a and common difference/ratio
2. Determine if AP (constant difference) or GP (constant ratio)
3. Find nth term using formula
4. Find sum using sum formula
5. For GP with |r| < 1: check if infinite sum needed

## Common JEE Tricks
- For max/min of ax+b/cx+d type: use AM-GM
- For |x-a| + |x-b|: use geometric interpretation (distance)
- For trig equations: use general solution form
- For probability with symmetry: use complementary counting
- For det of special matrices: use properties (triangular, symmetric)
