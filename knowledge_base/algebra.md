# Algebra - Core Formulas & Identities

## Quadratic Equations
- Standard form: ax² + bx + c = 0
- Quadratic formula: x = (-b ± √(b²-4ac)) / 2a
- Discriminant: D = b² - 4ac
  - D > 0: two distinct real roots
  - D = 0: one repeated real root
  - D < 0: two complex conjugate roots
- Sum of roots: α + β = -b/a
- Product of roots: αβ = c/a

## Factoring Identities
- (a+b)² = a² + 2ab + b²
- (a-b)² = a² - 2ab + b²
- (a+b)(a-b) = a² - b²
- (a+b)³ = a³ + 3a²b + 3ab² + b³
- (a-b)³ = a³ - 3a²b + 3ab² - b³
- a³ + b³ = (a+b)(a² - ab + b²)
- a³ - b³ = (a-b)(a² + ab + b²)

## Arithmetic & Geometric Progressions
- AP: aₙ = a + (n-1)d
- AP Sum: Sₙ = n/2 × [2a + (n-1)d]
- GP: aₙ = a·rⁿ⁻¹
- GP Sum (finite): Sₙ = a(rⁿ-1)/(r-1) for r≠1
- GP Sum (infinite, |r|<1): S = a/(1-r)

## Inequalities
- If a > b and c > 0: ac > bc
- If a > b and c < 0: ac < bc
- AM-GM: (a+b)/2 ≥ √(ab) for a,b ≥ 0
- AM-GM-HM: AM ≥ GM ≥ HM
- Cauchy-Schwarz: (a₁²+a₂²)(b₁²+b₂²) ≥ (a₁b₁+a₂b₂)²

## Logarithms
- logₐ(mn) = logₐm + logₐn
- logₐ(m/n) = logₐm - logₐn
- logₐ(mⁿ) = n·logₐm
- Change of base: logₐb = log b / log a
- logₐa = 1, logₐ1 = 0

## Absolute Value
- |x| = x if x ≥ 0, -x if x < 0
- |xy| = |x||y|
- |x+y| ≤ |x| + |y| (Triangle inequality)
- |x-y| ≥ ||x| - |y||

## Binomial Theorem
- (a+b)ⁿ = Σ C(n,k) aⁿ⁻ᵏ bᵏ for k=0 to n
- C(n,k) = n! / (k!(n-k)!)
- Middle term: if n is even, middle term is T(n/2+1)
- General term: T(r+1) = C(n,r) aⁿ⁻ʳ bʳ

## Functions
- Domain: set of all valid input values
- Range: set of all output values
- Composite: (f∘g)(x) = f(g(x))
- Inverse: f⁻¹(f(x)) = x (if f is bijective)
- Even function: f(-x) = f(x)
- Odd function: f(-x) = -f(x)

## Common Mistakes in Algebra
- Forgetting to flip inequality when multiplying by negative
- Squaring both sides can introduce extraneous solutions
- log(a+b) ≠ log(a) + log(b)
- √(a²+b²) ≠ a+b
- (a+b)² ≠ a² + b²
